# Launching the SSMS Extension Gallery

I’m excited to announce the launch of the **SSMS Extension Gallery**:

👉 https://erikej.github.io/SsmsExtensions/

This new page gives SQL Server developers and DBAs a quick overview of tools that can improve day-to-day work in SQL Server Management Studio.

Over the years I have also published several SSMS extensions myself, and you can find them here:

👉 https://www.vsixgallery.com/search/?q=ssms

The goal with this launch is simple: make it easier to discover both free and commercial SSMS extensions in one place.

If you build or maintain an SSMS extension and want it included, please open an issue or pull request.
